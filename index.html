Little-XHR

Light-weight XmlHttpRequest Object.

Download Download fork Little-XHR on GitHub

What is it?

This object is for cross-browser XMLHttpRequest handling. It is essentially a wrapper object which simplifies your use of the XmlHttpRequest object.

The XHR object provides a clean and easy way for doing asyncronous requests using the familiar API, leveraging success and failure callbacks while allowing you to pass JSON formatted data to the server.

API Example

xhr.get({
    url: "/path/to/some/page",
    success: function(xhrObject) { 
        console.log(xhrObject.responseText);
    },
    failure: function(xhrObject) { 
        console.log(xhrObject.status + ' - ' + xhrObject.statusText);
    },
    enctype: "multipart/form-data"
});

xhr.post({
    url: "/path/to/some/page",
    success: function(xhrObject) { 
        console.log(xhrObject.responseText);
    },
    failure: function(xhrObject) { 
        console.log(xhrObject.status + ' - ' + xhrObject.statusText);
    },
    enctype: "multipart/form-data",
    data: {"key1":"value","key2":"Hello World!"}
});

xhr.request("POST", {
    url: "/path/to/some/page",
    success: function(xhrObject) { 
        console.log(xhrObject.responseText);
    },
    failure: function(xhrObject) { 
        console.log(xhrObject.status + ' - ' + xhrObject.statusText);
    },
    enctype: "application/x-www-form-urlencoded",
    data: {"key1":"value","key2":"Hello World!"}
});

XHR2?

XMLHttpRequest Level 2 or XHR2 is simply the second specification to the XmlHttpRequest interface. This is a new specification and has not been fully adopted by all modern browsers. Consideration on supporting this will be reviewed once the specification has matured.
